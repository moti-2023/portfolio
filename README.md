# Bank & Design Portfolio

Professional graphic design portfolio showcasing bank designs, coffee branding, and fintech logos.

## Projects Included

- **SR Coffee** - Logo design with GIMP & Canva
- **Aimo Coffee** - Coffee brand logo with GIMP
- **MobiCash** - Fintech logo with Figma & Canva
- Bank branding and design examples

## Tools Used

- GIMP
- Figma
- Canva

## Setup Instructions

1. Add your logo images to the `images/` folder:
   - `sr-coffee.jpg`
   - `aimo-coffee.jpg`
   - `aimo-logo.jpg`
   - `mobicash.jpg`

2. Open `index.html` in your browser to view the portfolio

## Free Hosting Options

### Option 1: GitHub Pages (Recommended)
1. Create a GitHub account at https://github.com
2. Create a new repository named `portfolio`
3. Upload all files (index.html, styles.css, script.js, images folder)
4. Go to Settings > Pages
5. Select "main" branch and click Save
6. Your site will be live at: `https://yourusername.github.io/portfolio`

### Option 2: Netlify
1. Go to https://netlify.com
2. Sign up for free
3. Drag and drop your entire folder
4. Get instant live URL

### Option 3: Vercel
1. Go to https://vercel.com
2. Sign up with GitHub
3. Import your repository
4. Deploy automatically

### Option 4: Render
1. Go to https://render.com
2. Create free static site
3. Connect your GitHub repository
4. Deploy
